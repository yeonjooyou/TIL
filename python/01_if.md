### elif문

> else if
>
> if else문에서 조건을 만족하지 않을 때 세부 조건을 추가 점검

```python
age = 23
if age < 19 :
    print("애들은 가라.")
elif age < 25 :
    print("대학생 입니다.")
else :
    print("어서 옵쇼.")
```
