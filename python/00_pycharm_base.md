### 확장열(대행문자)
* \n (개행/줄바꿈)

  ```python
  print("ab\ncd\nef")
  ```

  > ab
  > cd
  > ef

* \t(탭)

  ```python
  print("ab\tcd\tef")
  ```

  > ab	cd	ef

* 긴 문자열 (행 단위 구분)

  ```python
  a = """aaa
  aaa
  aaa"""
  print(a)
  ```

  > aaa
  > aaa
  > aaa
